Surgimento da Internet

A internet surgiu a partir de um projeto militar dos Estados Unidos chamado ARPANET,
criado nos anos 60 para conectar computadores e garantir comunicação em caso de guerra.
Com o tempo, ela foi se expandindo e evoluindo até se tornar o que conhecemos hoje,
permitindo o compartilhamento de informações no mundo todo. O surgimento da web nos anos 90,
com o HTML e os navegadores, foi o que popularizou de vez o uso da internet pelas pessoas.

Como funciona o HTML

O HTML é a base de qualquer site. Ele funciona como a estrutura de uma página, organizando
o conteúdo com “blocos” chamados de tags. Essas tags dizem o que cada parte da página representa,
como títulos, parágrafos, imagens, links e listas. O navegador lê esse código e exibe tudo visualmente para o usuário.
Apesar de não ser uma linguagem de programação, o HTML é essencial para montar a parte visual e estrutural de um site.

O que são tags

Tags são palavras-chave usadas no HTML para marcar e organizar os elementos de uma página,
como títulos, parágrafos, imagens e links. Elas funcionam como instruções para o navegador entender
o que deve ser exibido e de que forma. Cada tag tem uma abertura e, na maioria das vezes, um fechamento,
e tudo que estiver entre elas será o conteúdo daquela parte da página.
É com as tags que a gente monta toda a estrutura de um site.

Html semântico

Voltar

HTML semântico é uma forma de escrever o código usando tags que têm significado claro sobre o conteúdo que elas envolvem.
Por exemplo, em vez de usar uma tag genérica pra tudo, a gente usa tags como "header" para o cabeçalho, "nav" para a navegação,
"main" para o conteúdo principal e "footer" para o rodapé. Isso deixa o código mais organizado, fácil de entender e acessível
tanto para desenvolvedores quanto para os navegadores e leitores de tela. Além disso, ajuda no SEO,
porque os mecanismos de busca entendem melhor a estrutura do site.

Cliente x Server

Voltar

A relação entre cliente e servidor é como uma conversa. O cliente é quem faz o pedido — por exemplo,
o navegador que você usa para acessar um site. Já o servidor é quem responde, enviando os dados
que o cliente pediu. Quando você digita um site e aperta Enter, o cliente manda uma requisição pro servidor,
que então devolve o conteúdo da página. Esse processo acontece em segundos, e é assim que a internet
funciona por trás dos panos: o cliente pede, o servidor entrega.
